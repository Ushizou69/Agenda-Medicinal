# Agenda-Medicinal

## Descrição

Agenda médica que serve para facilitar o controle sobre seu tempo, podendo administrar suas atividades físicas, também como suas dosagens medicinais e tendo as prioridades definidas em cada uma suas tarefas.  

## Funcionalidades

- Calcula quando comprar novo remédio baseado na quantidade pílulas que o usuário tem, 
- Agenda de coisas para fazer durante a semana/dia com notificação, 
- Lista de compras;

## Objetivo

Ajudar pessoas com sua organização medicinal de maneira simples, tal qual pessoas de idade, ou, muito atarefadas; tendo em vista também o aprendizado, aprofundando o conhecimento em POO (Programação Orientada a Objetos).
